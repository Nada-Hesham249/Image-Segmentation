# Image Thresholding & Segmentation System

A computer vision project focused on **image segmentation techniques** using both classical thresholding methods and unsupervised learning-based clustering approaches.

The system works on both **grayscale and color images** to produce accurate region-based segmentation.

---
# 🔹 Mode 1: Image Thresholding Techniques

This mode focuses on converting grayscale images into binary or multi-class segmented outputs using different thresholding strategies.
